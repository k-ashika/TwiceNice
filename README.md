# Twice Nice – Thrift Store Web Application

"Twice Nice" is a full-stack e-commerce website that allows users to buy thrifted clothing and accessories. It includes user and admin roles, real-time payment integration, return system, and more.

## 💻 Technologies Used

- Frontend: Angular
- Backend: Spring Boot
- Database: MySQL
- Authentication: JWT
- Payment Integration: Razorpay

## 👤 User Features

- Register and Login
- Browse Products by Category
- View Product Details
- Add to Cart and Wishlist
- Razorpay Checkout
- View Order History
- Download Invoices
- Return Requests
- Submit Reviews
- Update Profile

## 🧑‍💼 Admin Features

- Admin Login
- Dashboard with Charts
- Add/Edit/Delete Products
- Manage Users
- View and Update Orders
- Approve/Reject Returns
- Moderate Reviews
- Update Admin Profile
